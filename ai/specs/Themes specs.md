Islands UI is our current and only supported theme.
No need to support a "New UI" theme — only the Island theme is used.

This is how Main Window looks like:
https://www.figma.com/design/zKwabe7qCf1c0LFu93997q/Int-UI-Kit--Islands?node-id=7070-143445&t=V8DOXcjhgUO6bzb3-4

The New UI is the previous default. But we don't use it now.

## Theme Variants

- **Light** — `theme-light` CSS class
- **Dark** — `theme-dark` CSS class

Both use the Island visual style (rounded panels, gaps, gradient backgrounds).
